# 10分本番ポッドキャスト実装 - 要件定義書

## 📊 プロジェクト概要

### 🎯 目標
- **10分完全版ポッドキャスト**: 毎日配信される高品質マーケットニュース
- **完全自動化システム**: 記事取得→AI台本生成→音声合成→配信の無人化
- **プロ級品質**: 投資家・経営者向け実践的金融情報提供

### 📈 成果指標
- **配信品質**: 10分間（2600-2800文字）の自然な音声コンテンツ
- **情報価値**: 当日の重要市場情報を6記事で包括的にカバー
- **配信安定性**: 毎日JST 7:30の確実な自動配信
- **ユーザー体験**: LINEからワンクリックでアクセス可能

## 🔧 技術要件

### 🤖 AI台本生成システム
**AI Model強化**: 
- **現行**: Gemini 2.0 Flash Lite（高速・低コスト）
- **新規**: **Gemini 2.5 Pro**（最高品質台本生成）
- **理由**: 台本品質が音声コンテンツの価値を決定するため、最高モデル使用

**プロンプトエンジニアリング**:
- **金融専門性**: 市場分析・投資視点の深い洞察
- **聞きやすさ**: 専門用語解説・自然な話し言葉
- **実践性**: 投資家向けアドバイス・リスク評価

### 📊 データソース統合
**データベース構造**（masterブランチ最新版）:
```sql
-- Article テーブル（基本情報）
articles: id, url, title, body, source, published_at, scraped_at

-- AIAnalysis テーブル（拡張済み）
ai_analysis: 
  - summary, sentiment_label, sentiment_score
  - category (金融政策、経済指標、企業業績等)  ← 新規追加
  - region (japan, usa, china, europe, other)   ← 新規追加
  - analyzed_at, processing_time_ms
```

**記事選択アルゴリズム**:
```python
# 重要度スコア計算
score = (
    abs(sentiment_score) * 2.0 +        # センチメント重要度
    len(summary) / 100.0 +               # 要約詳細度  
    category_weight(category) * 1.5 +    # カテゴリ重要度（NEW）
    region_weight(region) * 1.0 +        # 地域バランス（NEW）
    hours_since_published * -0.1         # 新着度重視
)
```

### 🎙️ 音声生成システム
**Google Cloud TTS設定**:
- **音声**: `ja-JP-Standard-A` (女性声・自然)
- **品質**: LINEAR16, 24kHz（高音質）
- **速度**: 1.0倍速（聞きやすい自然な速度）
- **出力**: MP3形式、10-15MB以内

### 📡 配信システム
**LINE + GitHub Pages統合**:
- **GitHub Pages**: 音声ファイル公開・直接アクセス
- **LINE通知**: リッチメッセージでの配信通知
- **RSS配信**: ポッドキャストアプリ対応

## 📝 台本構成要件

### 🎯 10分間（2600-2800文字）構成

#### **1. オープニング** (30秒-1分, ~200文字)
- 親しみやすい挨拶
- 今日の日付・曜日
- 注目トピックの予告
- 聞き手への語りかけ

#### **2. メインコンテンツ** (8-8.5分, ~2200文字)
**記事1: 最重要ニュース** (400文字)
- 詳細解説・背景情報
- 市場への影響分析
- 専門用語の分かりやすい説明

**記事2: 重要市場動向** (400文字)  
- 投資家視点での分析
- セクター別影響評価
- 今後の展望

**記事3-4: 関連情報** (300文字 x 2)
- 地域別バランス（日本・米国・欧州等）
- 異なるカテゴリーからの選択
- 相互影響の解説

**記事5-6: 補完情報** (200文字 x 2)
- 簡潔な紹介・要点整理
- 他ニュースとの関連性

**総合分析・まとめ** (400文字)
- 本日の市場全体動向
- 注意すべきリスク要因
- 投資家へのアドバイス（推奨は避ける）

#### **3. クロージング** (30秒-1分, ~200文字)
- 本日のまとめ
- 明日以降の注目事項予告
- 感謝の言葉・次回予告

### 🎨 表現指針

**必須要素**:
- **話し言葉**: 「〜ですね」「〜ますから」等の自然な表現
- **専門用語解説**: 「FRB（エフアールビー・米連邦準備制度理事会）」等
- **数値の読み方**: 「1兆2,500億円」→「1兆2500億円」等
- **適切な間**: 句読点での自然な区切り

**避けるべき要素**:
- 30文字を超える長文（自動分割対象）
- 投資推奨・断定的予測
- 感情的すぎる表現
- 複雑な専門用語の連続使用
