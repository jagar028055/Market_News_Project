# ポッドキャスト機能デプロイチェックリスト

このドキュメントは、ポッドキャスト機能を本番環境にデプロイする際の確認事項をまとめています。

## 事前準備

### 1. 環境変数の設定

#### GitHub Secrets 設定
以下の環境変数をGitHubリポジトリのSecrets設定に追加してください：

**必須設定:**
- [ ] `GEMINI_API_KEY` - Gemini API キー
- [ ] `GOOGLE_SERVICE_ACCOUNT_JSON` - Google サービスアカウントJSON
- [ ] `GOOGLE_DRIVE_OUTPUT_FOLDER_ID` - Google Drive 出力フォルダID
- [ ] `LINE_CHANNEL_ACCESS_TOKEN` - LINE チャンネルアクセストークン
- [ ] `LINE_CHANNEL_SECRET` - LINE チャンネルシークレット

**ポッドキャスト固有設定:**
- [ ] `ENABLE_PODCAST_GENERATION=true`
- [ ] `PODCAST_RSS_BASE_URL` - GitHub PagesのURL
- [ ] `PODCAST_RSS_TITLE` - ポッドキャストタイトル
- [ ] `PODCAST_RSS_DESCRIPTION` - ポッドキャスト説明
- [ ] `PODCAST_AUTHOR_NAME` - 著者名
- [ ] `PODCAST_AUTHOR_EMAIL` - 著者メールアドレス

**オプション設定:**
- [ ] `PODCAST_MONTHLY_COST_LIMIT` - 月額コスト制限（デフォルト: 10.0）
- [ ] `PODCAST_TARGET_DURATION_MINUTES` - 目標再生時間（デフォルト: 10.0）
- [ ] `PODCAST_MAX_FILE_SIZE_MB` - 最大ファイルサイズ（デフォルト: 15）

## トラブルシューティング

### よくある問題と対処法

#### 1. TTS 生成エラー
**症状:** 音声生成が失敗する
**対処法:**
- [ ] Gemini API キーの確認
- [ ] API クォータ制限の確認
- [ ] 文字数制限の確認
- [ ] 特殊文字の除去

#### 2. 音声処理エラー
**症状:** 音声ファイルの処理が失敗する
**対処法:**
- [ ] アセットファイルの存在確認
- [ ] ファイル形式の確認
- [ ] 音声ファイルの破損チェック
- [ ] 一時ディスクスペースの確認

#### 3. RSS 生成エラー
**症状:** RSS フィードが生成されない
**対処法:**
- [ ] 出力ディレクトリの権限確認
- [ ] メタデータの完全性確認
- [ ] XML 形式の妥当性確認

#### 4. LINE 配信エラー
**症状:** LINE メッセージが送信されない
**対処法:**
- [ ] アクセストークンの確認
- [ ] API レート制限の確認
- [ ] メッセージ形式の確認
- [ ] ネットワーク接続の確認

#### 5. GitHub Actions エラー
**症状:** ワークフローが失敗する
**対処法:**
- [ ] 環境変数の設定確認
- [ ] 依存関係の確認
- [ ] タイムアウト設定の調整
- [ ] ログの詳細分析

## 緊急時の対応

### 配信停止が必要な場合
1. [ ] GitHub Actions ワークフローの無効化
2. [ ] `ENABLE_PODCAST_GENERATION=false` の設定
